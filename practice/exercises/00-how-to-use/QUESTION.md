# How to use this practice folder

This is not an exam task. Read [../../README.md](../../README.md) in `practice/`.

## Workflow

1. Open only **`QUESTION.md`** for an exercise (English).
2. Work on the target VM with `vagrant ssh server1` or `server2`.
3. Run **`./scripts/lab-practice.sh validate NN`** when done.
4. Read **`VALIDATION.md`** (Portuguese) only if validation fails.

## Ask the assistant

You can say: *"Give me exercise 01 without spoilers"* or *"Validate exercise 02"* — validation feedback may be in Portuguese.
